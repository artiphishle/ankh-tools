# ankh-uml

Generation of plantUML (& soon Mermaid) diagrams:
  
- Class Diagram
- Sequence Diagram (WIP)

## Quickstart


```bash
# Class Diagram
npx ankh-uml src/examples/tests/uml-base-subclass/User.ts .

# Sequence Diagram
npx ankh-uml src/examples/tests/uml-base-subclass/User.ts . --sequence
```
